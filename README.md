# Claude Code Profiles

用于管理 Claude Code 配置文件的桌面应用程序。

## 功能特性

- 📁 **配置管理**：创建、编辑、删除、切换多个配置文件

## 快速开始

### 下载使用
1. 下载 `claude-code-profiles.exe`
2. 双击运行即可

### 从源码构建
```bash
git clone https://github.com/yourusername/claude-code-profiles.git
cd claude-code-profiles
cargo build --release
```

## 技术栈

- **Rust** + **egui** - 跨平台GUI框架
- **serde** - JSON处理
- **rfd** - 文件对话框
